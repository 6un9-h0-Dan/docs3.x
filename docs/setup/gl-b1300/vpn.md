# VPN

GL.iNetルーターは、OpenVPNとWireGuardのVPNサーバー・クライアントがインストールされています。

詳細な設定手順については、以下のリンクをご参照してください：

- [**OpenVPN**](https://docs.gl-inet.com/jp/3/app/openvpn/)
- [**WireGuard**](https://docs.gl-inet.com/jp/3/app/wireguard/)
